:::note[HTTP Message Signatures]
HTTP Message Signatures protect the integrity of Open Payments API requests. The Open Payments TypeScript SDK includes the <LinkOut href='https://github.com/interledger/open-payments/tree/main/packages/http-signature-utils'>HTTP Signature Utils library</LinkOut>, which provides tools for working with HTTP Message Signatures.
:::
